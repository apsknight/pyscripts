Python Scripts for fetching data from JSON APIs using `json` and `urllib2` module.

### API Used:
	* Github API
	* numbersapi.com
	* Tronald Dump API
	* Talaikis Quotes API
	* fixer.io API

### How to Run:

```bash
git clone https://github.com/apsknight/pyscripts`  
cd pyscripts
python <script name>.py
```
